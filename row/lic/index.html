1. 名称

1-1. 本団体の名称は以下の通りとする。

  • 正式名称：StedTechOrganization
  • 表示名：StedTech, stedtech, stedtech-org, StedTechOrganization™, StedTech™
  • 日本語表記：ステッドテックオーガニゼーション（ステッドテック）

2. 制作物の利用について

2-1. 本団体が制作した作品・コンテンツ等（以下「作品等」）に関する著作権その他一切の権利は、本団体に帰属する。

2-2. 作品等の利用により生じた損害・トラブル等について、本団体は一切の責任を負わない。

2-3. 作品等の利用にあたっては、以下の個別規約を適用する。

個別規約 A

  • A-1. 商用利用・改造・再配布・自作発言を禁ずる。
  • A-2. 作品の制作意図や目的に反する利用を禁ずる。
  • A-3. 作品案内に記載された使用方法を遵守しなければならない。

個別規約 B

  • B-1. 商用利用を許可する。
  • B-2. 改造・再配布・自作発言を禁ずる。

個別規約 C

  • C-1. 商用利用および改造を許可する。
  • C-2. 再配布・自作発言を禁ずる。

個別規約 D

  • D-1. 第三者による使用を禁ずる。

3. ロゴおよび団体名の使用について

3-1. 本団体のロゴは、事前の許可なく使用することを禁ずる。

3-2. ロゴの使用を希望する場合は、本団体の公式SNS（X／旧Twitter）へ連絡すること。

3-3. 団体名については、原則として許可なく使用してよいものとする。ただし、以下の場合は例外とする。

  • 3-3-1. 本団体より使用停止の通告を受けた場合、速やかに使用を停止しなければならない。

4. 規約の変更について

4-1. 本利用規約は、2025年5月9日時点の内容である。

4-2. 本団体は、必要に応じて本規約を予告なく変更することがある。

4-3. 規約変更後は、変更前に作品等をダウンロード・利用した場合であっても、変更後の規約が適用されるものとする。

5. 管理

5-1. 本利用規約は、StedTechOrganization Admin div. によって適用・施行される。

5-2. 前項に定める者以外は、本規約を変更することはできない。
